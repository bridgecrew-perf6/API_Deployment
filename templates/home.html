  • Introduction

  • Welcome! This Webpage provides information regarding the use of the API. This API provides a price prediction for a given property with the help of machine learning model.

  • API

  • How do I use this API?

    You can use this API with a POST request to send a JSON object with all the required fields and their respective values. Please make sure that you accomplish all the requirements before sending your request(all string values arecase sensitive). You can find all requirements here.

    This API is built with Flask. Pydantic library is checking if the POST parameters are given correctly and provide an error message to the user with pointing the issue.

    {
    "errors": {
    "building_condition": "unexpected value; permitted: 'As new', 'Just renovated',
    'Good', 'To be done up', 'To renovate', 'To restore'" },
    "prediction": null
    }

    If your request meets all the requirements you will receive a predicted price for the given property like below example.
    {
    "error": null,
    "prediction": 316797.85
    }

  • Prediction Process

  • Data Source

    Data to train a machine learning model was from Immoweb in February with BeautifulSoup library.

  • Machine Learning Model

    As a first step data set was cleaned and we decided to choose certain features which seemed to be most correlated with the price.

    The second step all categorical values were converted to numerical values. After that Linear Regression model was selected and trained.

  • How good is the model?

    Unfortunately, the prediction is not that accurate at this moment. The predicted price is in acceptable range and kind of make sense but for the luxury houses our model makes so many mistakes. On a more technical level, the R2 score is about 0.73.
